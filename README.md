# Municipal-History
Boulder's Carnegie Library for Local History has a 5-document collection of municipal histories: https://localhistory.boulderlibrary.org/islandora/object/islandora%3A88836

These histories cover:

* 1871–1946
* 1965–1974
* 1975–1979
* 1980–1984
* 1985–1990

The PDF files were produced by using the "Reduce File Size" functionality in Adobe Acrobat.

The Markdown files were produced by using the Datalab.to
